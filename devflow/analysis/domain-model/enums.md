# Enumeraciones y Value Sets

Catálogo de **estados, códigos y conjuntos de valores** reutilizables por las
entidades del dominio.

---

## Formato

Cada enum se documenta con:
- **Nombre** — identificador del enum (referenciado desde las entidades)
- **Valores** — lista de valores posibles con su significado
- **Notas** — aclaraciones de uso
